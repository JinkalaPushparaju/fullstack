This project is a full-stack web application designed as a real-world solution with a public landing page and an admin panel. The application allows dynamic management of projects, clients, contact form submissions, and newsletter subscriptions.

The landing page fetches and displays data from the backend, while the admin panel provides full control over content management. The project follows clean code practices and a modular structure.
